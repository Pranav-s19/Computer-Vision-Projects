# Computer Vision Projects

This repository contains various computer vision projects and implementations. The projects are implemented using Python with popular computer vision and deep learning libraries.

## Environment Setup

The project uses a Conda environment with Python 3.11. To set up the environment:

```bash
# Create conda environment
conda create -n computer-vision python=3.11 -y

# Activate the environment
conda activate computer-vision

# Install requirements
pip install -r requirements.txt
```
